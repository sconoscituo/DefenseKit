## 보안 인가 절차

### 비밀취급인가 종류
- **II급 비밀**: 대부분의 방산 연구개발 직무에 필요
- **III급 비밀**: 일반 방산 업무

### 절차
1. **입사 후 신청**: 소속 기관장 추천으로 신청
2. **신원조회**: 경찰청 범죄경력 조회
3. **신원조사**: 국가정보원 또는 국군방첩사령부
4. **가족 관계 확인**: 직계가족 국적, 해외 거주 여부 확인
5. **인가 완료**: 통상 3~6개월 소요

### 주의사항
- 이중국적자, 직계가족 중 외국 국적자 있는 경우 심사 강화
- 해외 장기 거주 이력 있을 경우 추가 조회
- 범죄 전력 (음주운전 포함) 있는 경우 인가 제한 가능성

## 면접 준비 팁

### 기술 면접

**자주 출제되는 기술 문제:**
1. 칼만 필터 예측/업데이트 단계를 수식으로 설명하라
2. FFT의 시간복잡도는? Cooley-Tukey 알고리즘 설명
3. 실시간 시스템에서 메모리 동적 할당을 피해야 하는 이유
4. C++ RAII 패턴과 방산 임베디드에서의 활용
5. 탄도 계산에서 수치 적분법 선택 기준 (오일러 vs RK4)
6. 도플러 효과와 레이더 속도 측정 원리

**포트폴리오 발표 포인트:**
- 알고리즘의 수학적 배경 설명 능력
- 실시간 성능 최적화 경험
- MIL-STD / DO-178C 등 군 규격 인지도
- 수치 정밀도와 부동소수점 처리 이해

### 인성 면접
- **지원 동기**: 방산 산업의 국가 안보 기여, 기술적 도전
- **군 복무**: 관련 병과 경험이 있다면 적극 어필
- **보안 의식**: 기밀 정보 처리에 대한 책임감 강조
- **장기 근속 의지**: 방산 기업은 이직률 낮은 지원자 선호
